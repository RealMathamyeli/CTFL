## ‚ú® Hello World: Bringing Your First JavaScript Script to Life (approx. 45 minutes)

### üìÑ Embedding JavaScript in HTML
*   Create a new HTML file (e.g., `index.html`) in your project folder.
*   Add the basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
*   Learn the three common methods for embedding JavaScript into an HTML page:
    *   As an internal script in the `<head>` section.
    *   As an internal script before the closing `</body>` tag.
    *   As an external script (`<script src="your_script.js"></script>`).
*   Create an external JavaScript file (e.g., `script.js`) in the same folder.

### üöÄ Running and Testing Your Script
*   Add the command `console.log('Hello from the external script!');` to your `script.js` file.
*   Embed this external file in your `index.html` (before the closing `</body>` tag is often good practice).
*   Open your `index.html` in the browser and check the browser console to see if your message appears there.
*   Experiment with an internal script that, for example, triggers an alert dialog (`alert('Hello from an internal script!');`).
*   Compare the results and consider when each embedding method makes sense.
