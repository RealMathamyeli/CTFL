# Research Git and GitHub (Optional)

Do this exercise only if you have completed all the previous exercises and have extra time.

Answer the following questions in a well structured Markdown document:

- What is `Git`?
- What is `GitHub`?
- What is a `repository` in GitHub?
- What does '`cloning a repository`' mean?
- What is the difference between cloning a respository using `HTTPS` and `SSH`?
- Why do you think the instructor recommends using SSH over HTTPS for cloning repositories? [Use this Issue thread to help you answer the question](https://github.com/orgs/community/discussions/151462).
- Why is it important to configure a `username` and `email` after installing Git for the first time?
- Why is it important to own a `GitHub account` as an IT professional?
