### Exercise 2: Calculate Circle Area

Write a function `calculate_circle_area` that takes the radius as a parameter and returns the area of the circle. Use `3.14159` for PI.

**Formula:** Area = PI × radius²

**Example:**

```python
area = calculate_circle_area(5)
print(area)  # Should print: 78.53975
```

### Exercise 3: Temperature Converter

Create two functions:

- `celsius_to_fahrenheit(celsius)` - converts Celsius to Fahrenheit
- `fahrenheit_to_celsius(fahrenheit)` - converts Fahrenheit to Celsius

**Formulas:**

- Fahrenheit = (Celsius × 9/5) + 32
- Celsius = (Fahrenheit - 32) × 5/9

### Exercise 4: Check Even or Odd

Write a function `is_even` that takes a number as a parameter and returns `True` if the number is even, and `False` if it's odd.

**Example:**

```python
print(is_even(4))   # True
print(is_even(7))   # False
```

### Exercise 5: String Repeater

Create a function `repeat_string` that takes two parameters: a string and a number. The function should return the string repeated that many times, separated by spaces.

**Example:**

```python
result = repeat_string("Hello", 3)
print(result)  # Hello Hello Hello
```
