# GitHub Pages Workshop

In diesem Workshop lernst du,

- wie man eine Webseite mit GitHub Pages erstellt und veröffentlicht.
- GitHub Pages ermöglicht es dir, statische Webseiten direkt aus deinen GitHub-Repositories zu hosten.

- Wir werden die folgenden Themen behandeln:
  - Die Grundlagen der GitHub-Benutzeroberfläche
  - Einrichten eines GitHub-Repositorys für deine Webseite
  - Erstellen und Organisieren deiner Webseiten-Dateien
  - Veröffentlichen und Aktualisieren deiner Webseite
